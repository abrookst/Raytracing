# Raytracing
A raytracer built in c++.
Current Features:
- Difuse, Metal, and Dilectric materials
- Ability to render an image at any resolution, aspect ratio, and FOV to a PPM image
- Adjustable camera
- Anti-Aliasing
- Focus Blur
- Motion Blur
- BVH Optimization
- Textures
- Perlin Noise
- Light Objects
- Fog
